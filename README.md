# 🛍️ PHANTOM STORE - متجر فانتوم الإلكتروني

**Professional E-Commerce Platform | منصة تجارة إلكترونية احترافية**

---

## 📌 نبذة عن المشروع

**PHANTOM STORE** هو متجر إلكتروني احترافي متكامل يجمع بين تصميم عصري وتجربة تسوق سلسة. يوفر حلاً شاملاً لبيع المنتجات عبر الإنترنت مع إمكانيات متقدمة للإدارة والتحليل.

**PHANTOM STORE** is a comprehensive professional e-commerce platform combining modern design with seamless shopping experience. It provides a complete solution for selling products online with advanced management and analytics capabilities.

## 🚀 البدء السريع

### 1. استنساخ المشروع

```bash
git clone https://github.com/itsAAA1/-phantom-store.git
cd -phantom-store
```

### 2. فتح المتجر

```bash
# للفتح المباشر
open phantom-store-pro.html

# أو استخدم خادم محلي
python -m http.server 8000
# ثم ادخل: http://localhost:8000
```

### 3. التخصيص

قم بتعديل الملفات التالية:
- `phantom-store-pro.html` - الواجهة الأمامية
- `styles.css` - الأنماط (إن وجدت)
- `script.js` - الوظائف (إن وجدت)

## 💳 نظام الدفع

المتجر يدعم عدة طرق دفع:

   - دفع آمن وموثوق

   - تحويلات بنكية
